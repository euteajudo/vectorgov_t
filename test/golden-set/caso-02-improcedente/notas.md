# Caso 02 — Improcedência por álea ordinária

## O que este caso testa

A capacidade do sistema de **negar pedidos sem mérito**. Esse caso intencionalmente combina 3 problemas que petições reais frequentemente trazem:

1. **Fundamentação em variação ordinária** (oscilação de combustível é risco normal)
2. **Citação de lei revogada** (Lei 8.666/93 não vale mais para contratos novos pós-2024)
3. **Ausência de documentação probatória** (sem memória de cálculo, sem nota fiscal)

## Sinais de qualidade na análise

✅ Bom:
- Veredito `improcedente`
- Identifica explicitamente "álea ordinária"
- Aponta o erro da Lei 8.666/93
- Lista os documentos faltantes que poderiam (futuramente) mudar o resultado
- Recomenda comunicação clara ao contratado

❌ Ruim (alerta):
- Aceita "aumento do combustível" como fato superveniente (não é)
- Cita Lei 8.666/93 também (replicando o erro do contratado)
- Veredito `procedente` ou `parcialmente_procedente`

## Por que importa

**Capacidade de NEGAR** é tão importante quanto capacidade de aprovar. Se o sistema só sabe deferir pedidos, vira "máquina de gastar dinheiro público sem critério".
